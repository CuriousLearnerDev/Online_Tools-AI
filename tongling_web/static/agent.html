使用说明

  1. 在桌面「终端工作台」确认 HexStrike Server 已启动（绿色日志输出）。

  3. 在 Skills 页面 同步所需技能包。
  4. 在本机 Claude Code 终端（node_ai/claude-code）启动 claude 进行渗透任务。
